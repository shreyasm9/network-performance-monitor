# 📡 Network Performance Monitor

A simple real-time network monitoring tool that tracks:

- 📶 Ping (Latency)
- ⬇️ Download Speed
- ⬆️ Upload Speed

## 🚀 Features

- Real-time monitoring (every 30 seconds)
- Web dashboard using Flask
- Graph visualization using Chart.js
- CSV-based data storage (no database required)
- Lightweight and beginner-friendly

## 🛠️ Tech Stack

- Python (Flask, psutil, ping3, speedtest-cli)
- HTML + Chart.js
- CSV for logging
